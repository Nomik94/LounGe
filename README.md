## 주요 기능

### ⭐️ 그룹 창설 및 가입으로 소규모 커뮤니티 서비스 제공

- tag 검색을 통해 원하는 성격의 그룹 조회 기능 제공
- 뉴스피드 작성을 통한 커뮤니티 활성화
- 추후 업데이트를 통해 다양한 편의성 기능 제공 예정

### ⭐️ 캘린더를 통한 일정 관리 기능

- fullcalendar 라이브러리를 이용하여 캘린더 기능 제공
- 그룹 일정 공유 기능 제공
- 추후 업데이트를 통해 다양한 편의성 기능 제공 예정
